# 🎓 CourseHub - AI Powered E-Learning Platform

**CourseHub** is a cutting-edge, AI-powered Course Platform that makes learning interactive and futuristic.

This repository contains the **Frontend** application, built entirely on the modern **Angular 19** framework, delivering a fast, responsive, and glassmorphism-inspired UI/UX.

🔗 **Backend Repository**: [CourseHub Backend API (Node.js)](https://github.com/almas200/cp-backend)

## ✨ Key Features
- **Angular 19 Architecture:** State-of-the-art framework leveraging modern routing and fast rendering.
- **AI Chatbot Interface:** Beautiful UI allowing students to ask questions directly to Google's Gemini AI.
- **Matrix Admin HUD:** A futuristic, dark-themed dashboard for administrators to monitor courses.
- **Responsive Design:** Mobile-first, glassmorphism-themed styling for an exceptional user experience.

## 🛠️ Tech Stack
- **Framework:** Angular 19
- **Styling:** Vanilla CSS with Modern Glassmorphism
- **Language:** TypeScript

## 🚀 Getting Started
To run this project locally, make sure you also run the backend server.

```bash
# Clone this repository
git clone https://github.com/almas200/cp-frontend.git

# Install dependencies
npm install

# Run the development server
ng serve
```
